# Accident-Prevention-System-on-a-Railway
This is a project by Group 49 under supervision of MR. LUBEGA GERALD about an obstacle detection system onboard a Train that stops a moving Train, Flashes a LED alert and sends a SMS notification alert with the GSM when an obstacle on the railway is detected

## Problem Statement and Solution
Railway accidents continue to endanger passengers, pedestrians and cargo despite the growing need for accident prevention systems for rail transportation. Human error, vehicle collision and delayed communication and signaling significantly contribute to these accidents. This project presented a low cost Accident Prevention System for Railways using an Arduino based obstacle detection mechanism equipped with an ultrasonic sensor, a blinking LED indicator and a GSM module for alert notifications. Testing confirmed effective detection within specific ranges and reliable alert activation. This solution can greatly enhance railway safety especially in developing countries with limited automation. 

## Setup guide to use the project code
The project code was written with the Arduino IDE software as such ensure to download the Arduino IDE from google
- In the Arduino website select the OS type of your computer whether Linux, Mac or Windows and select between the leagcy version of 1.8... or the newer version of  2... to downlaod the Arduino IDE software on your computer
- Search for github on google
- On its search bar, search for this project and click on the colored button with ***<> Code*** to download the zipfolder with the project code file
- After downloading the project code file on your computer, use the Arduino IDE to open and run it
